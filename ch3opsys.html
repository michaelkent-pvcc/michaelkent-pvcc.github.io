Operating Systems Terminology

Term

Definition

Operating System

The software that controls the overall operation of the computer including file management, the user interface, and the program execution enviroment.

Job Queue

The order in which tasks are executed.

FIFO

Stands for First In - First Out, and represents the strucuture of the job queue, dealing with the oldest jobs first.

Interactive Processing

A medium, usually a terminal, that allows for a user to interact with submitted jobs or programs being actively executed.

Time-sharing

Allows for multiple users to execute jobs simultaneously.

Multiprogramming

A method of implementing time sharing in which each user gets a dedicated period of time over very short intervals. Akin to juggling.

Application Software

Consists of the programs for performing tasks particular to the machine's utilization.

System Software

Performs tasks that are common to computer systems in general.

Utility Software

Perfoms tasks that are fundamental to to computer installations but are not included in the OS.

Graphical User Interface

Provides a visual guide or interface to the user by depicting objects graphically.

Shell

The most basic user interface that only depicts objects in plain text.

Kernel

Software components that operate the most basic functions of a computer.

Device Driver

Software units that communicate with controllers to carry out operations with external devices.

File Manager

A kernel whose sole job is to organize and coordinate mass storage devices.

Memory Manager

A kernel tasked with coordinating a computer's main memory.

Virtual Memory

Through the use of file paging, virtual memory allocates a portion of a mass storage device to be used as memory.

Boot Strapping

The startup sequence in which computers initialize all of it's required programs.

ROM

Stands for Read Only Memory, and contains data that cannot be altered.

Deadlock

A stalemate that occurs when two proccess that rely on each other are unable to simultaneously boot.

Spooling

A method of circumventing Deadlock by spoofing a required resource until that resource is available.

Super User (Admin)

The highest privelege level within a computer, allowing for alteration of any setting or program without restriction.

Multi-factor Authentication

An additional layer of security that (typically) utilizes OTP or TOTP technology to reduce the risk of a leaked or bad password vulnerability.
